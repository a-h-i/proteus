proteus
=======

How to build:
============
  After you have installed all of the dependencies. Use `scons proteus` to build the proteus tool. `scons lib` to build libproteus, which contais the ASR code.
  
Dependencies:
============
- scons
- clang 3.3 or higher
- boost 1.54 or higher
- port-audio 2.0
- pocketsphinx
- sphinxbase
- libblas
- lapack
- libsamplerate
- libsndfile
